## Instruction

- 직접적인 지시 전까지 파일 수정 금지
- 작업 과정에서 모호한 사항이 있을 경우 멈추고 질문
- 코드 수정사항이 많을 경우, 별도의 markdown 파일로 계획 작성해 검토 지시. 해당 파일을 사용자가 수정해 계획 개선하도록 유도
- Unreal Project 의 경우
  - Blueprints, Input, Animation 등 사용자가 수동으로 수정해야 하는 부분이 있다면, 해당 부분을 명확히 설명
  - SOLID 규칙을 준수하여 ActorComponent 등을 활용
